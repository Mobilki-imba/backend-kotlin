# Trading App — Спецификация

Симулятор биржевой торговли. MVP. Документация разбита по командам.

## Структура

| Документ | Кому |
|----------|------|
| [00-overview-contracts.md](00-overview-contracts.md) | **Всем командам — обязательно** |
| [01-driver.md](01-driver.md) | Команда драйвера (C / Linux) |
| [02-backend-go.md](02-backend-go.md) | Команда Market Data (Go) |
| [03-backend-kotlin.md](03-backend-kotlin.md) | Команда Client BFF (Kotlin) |
| [04-android.md](04-android.md) | Команда мобильного приложения |

## Как читать

1. Сначала все читают `00-overview-contracts.md` — там описаны компоненты, граф зависимостей и контракты между ними.
2. Каждая команда работает по своему документу.
3. Контракты — единый источник истины. Изменения в контрактах согласуются между смежными командами.

## Граф зависимостей

```
Android  ──REST/WS──▶  BFF (Kotlin)  ──gRPC/Redis──▶  Market (Go)  ──/dev/quotes──▶  Driver (C)
```

## Скоуп MVP

- Регистрация и логин
- Главный экран с live-котировками
- Детали инструмента (график, стакан, метрики)
- Трейдинг-вью (создание лимитной и рыночной заявки)
- Портфель
- История сделок и активных заявок
